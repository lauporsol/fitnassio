Fitnassio

Fitnassio es una aplicación que surge con el objetivo de facilitar la experiencia del ejercicio físico de los usuarios. Nuestra aplicación sugiere tanto rutas interesantes para correr por las ciudades como diferentes y personalizadas rutinas y ejercicios para mejorar la forma física Además ofrece la posibilidad de consultar el clima y el observar el progreso.
